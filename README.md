# Simple E-Commerce Cart System (Python OOP)

## Project Overview
This project is a simple **E-Commerce Shopping Cart System** built using Python and Object-Oriented Programming (OOP).
The program allows users to view available products, add products to a cart, and see the total price of the items added to the cart.
It demonstrates how real shopping cart systems work in a simplified way using Python classes.

##  Features
- Display available products
- Add products to a shopping cart
- View items inside the cart
- Calculate total cart price
- Menu-driven interactive system

## Technologies Used
- Python
- Object-Oriented Programming (OOP)

## How the Program Works

### 1️ Product Class
The `Product` class represents an item in the store.
Each product has:
- **name**
- **price**
Example:
Laptop - 50000
Phone - 20000
Headphones - 2000

### 2️Cart Class
The `Cart` class manages shopping cart operations.
Functions inside the cart:
- **add_product()** → Adds a product to the cart
- **show_cart()** → Displays cart items and calculates the total price
- 
### 3️ Show Products
Displays the list of available products for the user.

### 4️ Add to Cart
The user selects a product number and the product is added to the cart.
Example output: Laptop added to cart

### 5️ View Cart
Displays:
- All items in the cart
- Total price of the cart

Example:
Items in Cart:
Laptop - 50000
Phone - 20000
Total Price: 70000

### 6️ Exit
Ends the shopping program.
